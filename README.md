## 기능 구현 목록
- POS 시작, 메인화면 출력 및 선택
- 주문등록을 위한 테이블 출력 및 선택
- 메뉴 출력 및 메뉴 선택, 메뉴 수량 입력
- 메뉴 등록에 따른 테이블 출력 상태 변화
- 단순 결제
- 단순 결제 진행 후 테이블 상태 초기화
- 치킨 메뉴 할인률 추가
- 현금결제 할인률 추가
- 주문, 결제 예외처리
